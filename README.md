## 🏆 Hackathon Objective Alignment
Synthesis.meme is the next generation of meme platforms, designed to cut through the saturation of the current market. We combine **Deep Market Sentiment Intelligence** with an **On-Demand AI Brand Forge**, delivering a professional, high-fidelity experience inspired by the clean, geometric aesthetic of the **Sui Network**. This project is optimized to win on Technical Implementation (30%), Innovation (30%), and Community Appeal (30%).

## 🌌 Core Features

### 1. Sentinel Radar (`/sentinel`)
An advanced, real-time surveillance dashboard that monitors the BNB Chain mempool. It uses AI heuristics to classify token deployments by risk, surfacing low-risk "alpha" while blocking known rug-pull signatures and honeypots. Click through to any token for a deep-dive security audit.

### 2. MemeForge AI Studio (`/studio`)
The ultimate "Idea-to-Brand" pipeline. Users provide a concept, and our AI (powered by **OpenAI GPT-4o**) generates a complete, launch-ready brand package:
- **Name & Ticker** 
- **Narrative Lore**
- **AI-Synthesized Logo** (via Pollinations.ai)
- **Marketing & Launch Strategies**

### 3. Vibe Report (`/report`)
A comprehensive crypto-sentiment dashboard. We leverage the **CoinGecko API** to pull real-time market data, which is then processed to create an "Executive Vibe Summary"—a high-level sentiment read on the overall meme ecosystem.

## 🛠 Tech Stack

- **Framework:** Next.js 16.2.3 (App Router) + React 19
- **Web3:** Wagmi + Viem + RainbowKit (BNB Chain Mainnet/Testnet ready)
- **Styling:** Custom CSS design system inspired by **Sui Network**. Features a clinical "Stark White" light mode and a deep "Oceanic Navy" dark mode.
- **AI:** OpenAI GPT-4o for narrative generation; Pollinations.ai for image synthesis.
- **Data:** CoinGecko Core API.
- **Theming:** `next-themes` for seamless hydration-safe mode switching.
